# Подсказки по GITу

* Создание репозитория 
```sh 
git init
```
* Добваление файла
```sh
git add
```
* Создание коммита + сообщение
```sh
git commit -m "_"
```
* Вывод на экран истории всех коммитов
```sh
git log
```
* Вывод истории коммитов в виде строки + урезанный хэш 
```sh
git log --oneline
```
